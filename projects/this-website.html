This Website

A digital garden, built together.

this site is a collaboration. Ethan has the ideas, the vision, and the editorial voice. Ultron handles the code, the structure, and the deployment. Ethan reviews everything, revises what doesn't feel right, and pushes the direction forward. neither one builds this alone.

how it started

Ethan was laying in bed at 2am and had the idea. he texted Ultron on Telegram: "what if we made a project showcase website." five minutes of back and forth over text, speccing out design direction, picking projects, choosing the vibe.

Ultron built the first version in about 15 minutes. 10 project articles, architecture diagrams, GitHub Pages deployment. then Ethan reviewed it on his phone and started directing changes. "make it white, not dark." "add a pill nav that morphs when you scroll." each revision pushed live within minutes.

that's the pattern for everything on this site. Ethan directs, Ultron builds, Ethan reviews and revises. rinse and repeat until it's right.

the digital garden

this isn't a static portfolio. it's a living document that grows as new projects ship. Ethan decides what gets added and how it's framed. Ultron writes the first drafts and handles the technical side. Ethan edits until it sounds like him.

the articles are written in Ethan's style: short, punchy, conversational, real numbers, no fluff. that voice comes from months of working together — Ultron has context on every project because he helped build most of them.

the stack

  • HTML/CSS, no frameworks
  • GitHub Pages from main branch
  • Excalidraw diagrams rendered to PNG
  • deployed via git push
  • idea to first version: ~20 minutes over Telegram
  • every change goes through Ethan's review

it's a simple site on purpose. the content is what matters, and the process of building it together is the point.
