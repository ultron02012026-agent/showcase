Hardware / AI

Golf Ball Retriever Rover

Raspberry Pi 5 autonomous rover that retrieves golf balls from putting practice. OpenCV runs locally at 30fps for ball detection, PID control loop handles navigation, and a servo-actuated scoop picks up the ball. Software was complete before the $233 Amazon order even arrived. The AI wrote the whole stack from a Reddit post about golf ball retrievers.

Digital Pantry

Meal planning system with a digital pantry tracking inventory across four zones: cupboard, fridge, freezer, and room. Say "what's for dinner?" and it suggests meals from what you actually have. Say "what do I need from the store?" and it diffs your pantry against your staples list, generates a Notion card with a checklist grouped by store.

Life / AI

The Library

Killed Audible in 30 minutes. The AI agent sourced the PDF, chunked it into chapters, ran each through ElevenLabs TTS, and served the audio as an RSS feed on port 8282 via a launchd service. Subscribe in Apple Podcasts and every book shows up as a podcast with chapters as episodes. 6 books in the library. Built while the human was at the gym.

Infrastructure

AI Engineering Manager

Ship features from the couch via Telegram. Text an idea, the agent asks clarifying questions, analyzes the codebase through a sub-agent, creates a GitHub issue with an implementation plan, spawns an isolated Claude Code session to write the code, opens a PR into staging, and notifies you when it's ready. The orchestrator never reads code directly. Ralph Loops pattern: fresh instance per task, test, commit, repeat.
